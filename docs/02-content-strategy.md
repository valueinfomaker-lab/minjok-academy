## 메시지 우선순위 정리

학부모가 사이트 첫 진입 시 5초 안에 받아야 할 메시지 (메인 페이지 Above-the-Fold):

1. **"무엇을 하는 곳인가"** → 영어 어학원
2. **"누구를 위한 곳인가"** → 유아·초등 중심 (선택지에 따라 중·고등)
3. **"왜 믿을 만한가"** → 16년, 남가좌·북가좌 지역의 명가
4. **"다음에 무엇을 해야 하는가"** → 무료 상담 신청

이 4가지가 Hero 섹션 + 신뢰 지표 섹션 안에 모두 담겨야 한다.
