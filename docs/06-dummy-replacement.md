# 06. 더미 데이터 교체 작업 리스트

> 현재 사이트 코드에 박혀 있는 모든 더미값/플레이스홀더 사진의 **단일 출처 (single source of truth)** 리스트.
> 학원으로부터 정보를 수집하고, 개발자가 한 번에 교체할 때 이 문서를 기준으로 작업.

**사용법**
1. 학원 담당자에게 "A. 텍스트 정보" 표를 그대로 전달해서 빈칸을 채워 받기
2. 학원에서 "B. 사진 자산" 항목별로 사진을 받기 (드라이브/메일)
3. 받은 자료를 표시된 파일/줄 위치에서 교체

## D. 작업자 가이드 (개발자용)

### 더미 검색 명령

전체 더미가 한눈에 보이도록 grep:

```bash
# 텍스트 더미 (○, XXX-XXXX)
grep -rn "○\|XXX-XXXX" --include="*.html" .

# 사진 더미 (picsum.photos)
grep -rn "picsum.photos" --include="*.html" .

# 더미 이메일/카카오
grep -rn "info@minjok\|@minjok-eng" --include="*.html" .

# 가상 강사명
grep -rn "Ms\. Sarah\|Mr\. John\|김○○\|이○○" --include="*.html" .
```

### 교체 순서 권장

1. **Phase 1 (A-1 + A-2)** — 학원 기본 정보 + 사업자 정보. partial 2개 + 페이지 5개에 prevalent. 가장 큰 효과.
2. **Phase 2 (A-3 + A-5)** — 원장 인사말 + 강사진. About 페이지가 채워짐.
3. **Phase 3 (A-6 + A-8)** — 프로그램 상세 + 위치 정보. 서브 페이지 정합성.
4. **Phase 4 (A-4 + A-7)** — 연혁 + 후기. 신뢰 강화 요소.
5. **Phase 5 (B 전체)** — 사진 일괄 교체. picsum URL을 로컬 `assets/img/...` 경로로 변경.

### 사진 교체 시 권장 작업 흐름

1. 받은 사진을 `assets/img/<category>/<id>.webp` 로 저장 (WebP 변환 시 용량 ↓)
2. HTML의 `src="https://picsum.photos/seed/<id>/..."`를 `src="assets/img/<category>/<id>.webp"`로 교체
3. alt 텍스트도 실제 사진 내용에 맞게 수정
