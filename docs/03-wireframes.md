## 공통 — 헤더 / 푸터

### 헤더 (PC, 1440px)

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                                                                              │
│   [민족 어학원]    학원소개   프로그램▾   캠퍼스   상담·오시는길    ☎ 02-XXX-XXXX  [무료 상담]
│   (logo)                                                                     │
└─────────────────────────────────────────────────────────────────────────────┘
                                          │
                          (프로그램 hover ↓)
                          ┌────────────────────┐
                          │  유치부 (6층)       │
                          │  초등부             │
                          │  중·고등부          │
                          └────────────────────┘

- 높이: 80px
- 배경: 화이트 + 하단 1px 그레이 보더
- 스크롤 시 약간의 그림자 추가
- 로고: 좌측, 클릭 시 메인
- [무료 상담]: 골드 배경 + 흰색 텍스트, 우측 끝
```

## 반응형 브레이크포인트

| 디바이스 | 너비 | 적용 |
|---|---|---|
| Mobile | ~ 767px | 1단 레이아웃, 햄버거 메뉴, 하단 고정 CTA |
| Tablet | 768px ~ 1023px | 2단 레이아웃, 햄버거 유지 또는 압축 메뉴 |
| Desktop | 1024px ~ | 풀 네비게이션, 다단 그리드 |
| Wide | 1440px ~ | 최대 컨테이너 너비 1280px로 제한, 좌우 여백 |

## 컨테이너 / 그리드 규칙

- 최대 컨테이너: **1280px** (이상은 좌우 자동 여백)
- 좌우 패딩: 모바일 16px / 태블릿 24px / 데스크탑 32px
- 그리드: 12 컬럼, 거터 24px
- 섹션 세로 패딩: 모바일 48px / 데스크탑 96px

## 인터랙션 메모

| 요소 | 인터랙션 |
|---|---|
| Hero CTA | hover 시 살짝 위로 + 그림자 |
| 프로그램 카드 | hover 시 카드 전체가 살짝 떠오름 (translateY -4px) |
| 갤러리 | 좌우 화살표 + 드래그 스와이프 / 모바일은 가로 스크롤 |
| 후기 카드 | 자동 슬라이드 (5초) + 좌우 버튼 |
| 헤더 | 스크롤 시 하단 그림자 등장 |
| 플로팅 상담 버튼 | 첫 화면에서는 숨김, 200px 스크롤 후 등장 (fade-in) |
| 폼 제출 후 | 페이지 이동 없이 토스트 메시지 또는 인라인 성공 표시 |
