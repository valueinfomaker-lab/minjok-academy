# 01. 사이트맵 & 메뉴 구조

> 민족 어학원 사이트의 전체 페이지 구조, 메뉴 트리, URL 설계.

---

## 1. 사이트맵 (전체 트리)

```
민족 어학원 (Home: /)
│
├─ 학원소개 (/about)
│   ├─ 인사말 (#greeting)
│   ├─ 16년 연혁 (#history)
│   ├─ 교육철학 (#philosophy)
│   └─ 강사진 (#teachers)
│
├─ 프로그램 (/programs)
│   ├─ 유치부 — 6층 유치관 (/programs/kindergarten)
│   ├─ 초등부 — 5층 초중등관 (/programs/elementary)
│   └─ 중·고등부 (/programs/secondary)
│
├─ 캠퍼스 (/campus)
│   ├─ 6층 유치관 시설 (#floor6)
│   ├─ 5층 초중등관 시설 (#floor5)
│   └─ 활동 갤러리 (#gallery)
│
├─ 상담·오시는길 (/contact)
│   ├─ 무료 상담 신청 (#consult)
│   └─ 위치·오시는길 (#location)
│
└─ (Footer 영역)
    ├─ 개인정보처리방침 (/privacy)
    └─ 사이트맵 (/sitemap)
```

> **참고**: RISE는 글로벌 프랜차이즈 체계라 5개 대메뉴(Who We Are / Program / Solution / Brand / Find Campus)이지만, 민족 어학원은 단일 캠퍼스이므로 4개 대메뉴로 단순화한다.

## 2. 메뉴 트리 (헤더 네비게이션)

### PC — 1단 + 드롭다운

```
┌─────────────────────────────────────────────────────────────────┐
│  [민족 어학원]   학원소개 │ 프로그램 ▾ │ 캠퍼스 │ 상담·오시는길   │ ☎ 02-XXX-XXXX │
└─────────────────────────────────────────────────────────────────┘
                              │
                              └─ 유치부 (6층 유치관)
                                 초등부 (5층 초중등관)
                                 중·고등부
```

- **드롭다운은 "프로그램"에만 적용**. 학원소개·캠퍼스·상담은 단일 페이지 내 앵커로 연결.
- 우측 상단에 **전화번호** + **무료 상담 버튼** (모바일은 햄버거 메뉴 + 하단 고정 CTA)

### 모바일 — 햄버거 + 하단 고정 CTA

```
┌────────────────────────────┐
│  [민족]                  ☰ │
├────────────────────────────┤
│  (햄버거 클릭 시 풀스크린)   │
│   학원소개                  │
│   프로그램          ▾      │
│     └ 유치부                │
│     └ 초등부                │
│     └ 중·고등부             │
│   캠퍼스                   │
│   상담·오시는길             │
│   ─────────                 │
│   ☎ 02-XXX-XXXX             │
└────────────────────────────┘

[화면 하단 고정] 📞 전화 상담  |  📝 상담 신청
```

## 3. 페이지 목록 (총 7페이지)

| # | 페이지명 | URL | 우선순위 | 비고 |
|---|---|---|---|---|
| 1 | 메인 | `/` (index.html) | **P0** | 디자인 시스템 확정용. 가장 먼저 구현 |
| 2 | 학원소개 | `/about.html` | P1 | 4개 섹션 (인사말/연혁/철학/강사진) |
| 3 | 프로그램 — 유치부 | `/programs/kindergarten.html` | P1 | 6층 유치관 상세 |
| 4 | 프로그램 — 초등부 | `/programs/elementary.html` | P1 | 5층 초중등관 초등 트랙 |
| 5 | 프로그램 — 중고등부 | `/programs/secondary.html` | P2 | 내신·특목고 |
| 6 | 캠퍼스 (시설 안내) | `/campus.html` | P2 | 시설 사진 + 활동 갤러리 |
| 7 | 상담·오시는길 | `/contact.html` | P1 | 폼 + 지도 + 운영시간 |

**우선순위 운영**:
- **P0** = MVP (최소 기능 사이트로 1차 오픈 가능)
- **P1** = 1차 오픈 시 반드시 포함
- **P2** = 1차 오픈 후 추가 가능

## 4. URL 설계 원칙

- **정적 사이트**이므로 모든 페이지는 `.html` 확장자 명시.
- 폴더 구조는 의미 단위로 묶되, 1단계 이상 깊어지지 않도록 한다 (`/programs/kindergarten.html`까지만).
- 페이지 내 섹션 이동은 **앵커(`#`)** 활용.
- 추후 동적 사이트로 확장 시 깨끗한 URL(`/programs/kindergarten`)로 자연스럽게 마이그레이션 가능한 구조.

```
파일 시스템 구조
└─ /
   ├─ index.html
   ├─ about.html
   ├─ campus.html
   ├─ contact.html
   ├─ privacy.html
   ├─ sitemap.html
   └─ programs/
      ├─ kindergarten.html
      ├─ elementary.html
      └─ secondary.html
```

## 5. 글로벌 공통 요소

전 페이지 공통으로 들어가는 요소:

### 헤더
- 로고 (좌측, 클릭 시 메인 이동)
- 네비게이션 (중앙/우측)
- 전화번호 (PC 우측 상단)
- 무료 상담 버튼 (PC 우측 상단, 골드 컬러)
- 햄버거 메뉴 (모바일)

### 푸터
- 학원 정보 (상호, 사업자번호, 대표자, 주소, 전화, 이메일)
- 사이트맵 (간략)
- 운영시간 (유치부 09:00~15:00, 초중등부 평일 오후~저녁)
- SNS 링크 (인스타그램, 블로그, 카카오톡 채널) — 보유 시
- 카피라이트

### 플로팅 요소
- **무료 상담 신청** 플로팅 버튼 (우측 하단, 골드)
- **카카오톡 채팅** 플로팅 버튼 (보유 시)
- **맨 위로(↑)** 버튼 (스크롤 시 등장)

## 6. 네비게이션 시나리오 (사용자 흐름)

### 시나리오 A: "유치원 알아보러 온 학부모"
```
구글 검색 → /  (메인)
              ↓ 신뢰 지표 + 연령별 프로그램 카드 확인
              ↓ "유치부 더 보기" 클릭
        /programs/kindergarten
              ↓ 6층 유치관 사진 + 일과표 확인
              ↓ "무료 상담 신청" 버튼 클릭
        /contact (또는 모달)
              ↓ 폼 작성 → 완료
```

### 시나리오 B: "초등 학부모, 학원 비교 중"
```
지인 추천 → /
              ↓ 16년 연혁 + 강사진 신뢰 지표 확인
              ↓ 메뉴 "프로그램 → 초등부"
        /programs/elementary
              ↓ 수준별 분반 / 커리큘럼 / 시간표 확인
              ↓ 메뉴 "학원소개 → 강사진"
        /about#teachers
              ↓ 강사 이력 확인
              ↓ 전화 또는 카카오톡 상담
```

### 시나리오 C: "특목고 진학 정보 찾는 중3 학부모"
```
구글 검색 "남가좌 특목고 영어" → /
              ↓ 메뉴 "프로그램 → 중·고등부"
        /programs/secondary
              ↓ 진학 실적 + 커리큘럼 확인
              ↓ 상담 신청
```

## 7. SEO 키워드 매핑 (페이지별)

| 페이지 | 타겟 키워드 |
|---|---|
| 메인 | 남가좌 영어학원, 북가좌 영어학원, 서대문 영어유치원 |
| 유치부 | 남가좌 영어유치원, 북가좌 영어유치부, 서대문 5세 영어 |
| 초등부 | 남가좌 초등 영어, 북가좌 영어학원 초등, 서대문 초등 영어학원 |
| 중·고등부 | 남가좌 특목고 영어, 북가좌 내신 영어, 서대문 중등 영어학원 |
| 학원소개 | 민족 어학원, 민족 영어학원 |
| 캠퍼스 | 민족 어학원 시설, 남가좌 영어유치원 시설 |

> **참고**: URL의 `utm_source=chatgpt.com` 같은 파라미터는 외부 광고/추천 트래픽 추적용. 자체 SEO에는 영향 없음.
